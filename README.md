# MiniShop - Website demo AEO và AIEO

Đây là website thương mại điện tử đơn giản dùng cho phần không bắt buộc của môn Thương mại điện tử.

## Nội dung website

- Trang chủ: giới thiệu website và kỹ thuật AEO/AIEO đã áp dụng.
- Trang sản phẩm: danh sách sản phẩm demo.
- Trang FAQ: câu hỏi thường gặp để hỗ trợ AEO.
- Trang liên hệ: thông tin liên hệ cơ bản.

## Kỹ thuật đã áp dụng

- Nội dung dạng câu hỏi - câu trả lời.
- FAQPage Schema.
- Product Schema.
- Organization Schema.
- File robots.txt.
- File sitemap.xml.
- Giao diện responsive đơn giản.

## Cách chạy

Mở file `index.html` bằng trình duyệt.

## Cách hosting bằng GitHub Pages

1. Tạo repository trên GitHub.
2. Upload toàn bộ file trong thư mục này lên repository.
3. Vào Settings > Pages.
4. Chọn Branch: main và Folder: /root.
5. Bấm Save.
6. Sau khi có link thật, thay `https://your-username.github.io/aeo-aieo-minishop-simple/` trong robots.txt, sitemap.xml và schema bằng link GitHub Pages thật.
